# 📊 SQL Sales Analysis

## Overview
This project analyzes retail sales data using SQL to extract key business insights related to revenue, profitability, and discount impact.

## Tech Stack
- SQL (SQLite)
- VS Code
- CSV Dataset

## Key Analysis Areas
- Revenue & Profit
- Profitability Segmentation
- Discount Impact
- Loss Analysis

## Key Insights
- Profitability is highly sensitive to discount levels.
- Revenue generation does not guarantee profitability.
- A large portion of orders contributes minimal or negative value.
- Pricing strategy appears to prioritize volume over margin.
